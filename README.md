## Mission

Create a backend server capable of supporting the front-end application of On/board.

## Technologies Used 

- Ruby on Rails
- Postgresql
- Github
- Heroku (back-end)
- Netlify (front-end)

## MVP / Achieved Targets

- Allow full CRUD functionality
- Create User Authentication / Authorization
- Multiple Models
- Utilize Postgresql
